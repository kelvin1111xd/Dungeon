點擊Dungeon.exe即可遊玩

玩家扮演法師(hp 3, atk 1)
WASD可移動，方向鍵可發射火球(可斜射，共8方向)
按鍵E與物品互動

按鍵1可保存記錄點（存檔check_point.txt）
按鍵2可讀取記錄點（讀取check_point.txt）

物品:
紅色小藥水： 補1血
紅色大藥水： 補2血
藍色小藥水： 加1攻擊力
藍色大藥水： 加2攻擊力
寶箱： 隨機掉落以上物品
梯子： 通往下一關

共3關卡，在第3關打死所有怪物即遊戲結束

使用素材：
https://pixel-poem.itch.io/dungeon-assetpuck
https://0x72.itch.io/dungeontileset-ii
